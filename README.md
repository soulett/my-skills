# My Skills

## Included Skills

### `paper-to-project-tech-note`

Turn a paper PDF or research-style technical document into a long-form project teaching note for beginners.

Main traits:

- de-paperized output by default
- beginner-friendly prerequisite teaching
- 4-6 core technical modules explained in depth
- thick deep-dive output is mandatory for detailed requests
- includes interview and presentation-ready phrasing

Repository path:

- `paper-to-project-tech-note/`

Core files:

- `paper-to-project-tech-note/SKILL.md`
- `paper-to-project-tech-note/references/de-paperization.md`
- `paper-to-project-tech-note/references/longform-structure.md`
- `paper-to-project-tech-note/agents/openai.yaml`
