# Resource Priority

## Priority Order

1. Local source materials
2. Official documentation or official course pages
3. Domestic Chinese video/course platforms
4. Domestic Chinese technical articles
5. Other community links

## Preferred Domestic Sources

- bilibili
- icourse163 / 中国大学MOOC
- 腾讯云开发者
- CSDN
- 掘金

## Selection Rule

If two resources cover nearly the same topic, keep the one that is:
- easier to access
- more clearly titled
- more teachable
- better aligned with the learner stage
