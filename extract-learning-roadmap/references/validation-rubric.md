# Validation Rubric

A resource is valid only if all checks pass.

## 1. Reachability

- the page opens now
- no obvious deletion page
- no obviously broken redirect

## 2. Topic Consistency

- the visible title and page topic match the intended section
- not just keyword overlap

## 3. Usefulness

- directly teaches the topic, or
- is a strong practical demo / course tied to the topic

## 4. Deduplication

- if several resources are similar, keep only the best few

## 5. Stage Fit

- beginner stages: prefer overview, intuition, and basics
- implementation stages: prefer code, workflow, and practical demos
- advanced stages: prefer theory, optimization, and deeper references
